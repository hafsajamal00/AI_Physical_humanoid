# Quickstart: Add Module 2 to Docusaurus Site

**Feature**: 002-digital-twin-module
**Date**: 2025-12-26
**Purpose**: Step-by-step instructions to add Module 2 structure to existing Docusaurus site

---

## Prerequisites

- Module 1 Docusaurus site already set up and running (from `specs/001-ros2-module/quickstart.md`)
- Node.js 18+ and npm installed
- Docusaurus dev server works (`cd book_frontend && npm start`)
- Git repository initialized

**Verify Installation**:
```bash
cd book_frontend
npm start  # Should open localhost:3000 with Module 1 content
```

---

## Step 1: Create Module 2 Directory Structure

### 1.1 Create Directories

From the repository root (`C:\Users\laiba\Desktop\AI_Book\`), run:

```bash
mkdir -p book_frontend/docs/module-2
mkdir -p book_frontend/static/img/module-2
```

### 1.2 Create Module 2 Category Configuration

Create `book_frontend/docs/module-2/_category_.json`:

```json
{
  "label": "Module 2: The Digital Twin (Gazebo & Unity)",
  "position": 2,
  "link": {
    "type": "generated-index",
    "description": "Learn digital twin concepts, Gazebo physics simulation, and Unity visualization for humanoid robots."
  }
}
```

**What this does**:
- Creates "Module 2" category in sidebar after Module 1
- Sets position (order in navigation: 2, after Module 1)
- Generates index page with description

## Step 3: Update Sidebar Configuration (Optional)

**Note**: Docusaurus autogenerated sidebar (configured in Module 1 setup) will automatically detect Module 2 directory and add it to navigation. No changes needed to `sidebars.js` unless you want manual control.

If you want manual sidebar control, edit `book_frontend/sidebars.js`:

```javascript
module.exports = {
  tutorialSidebar: [
    'intro',
    {
      type: 'category',
      label: 'Module 1: ROS 2 Robotic Nervous System',
      link: {
        type: 'generated-index',
        description: 'Learn ROS 2 as middleware for humanoid robotics.',
      },
      items: [
        'module-1/chapter-1-ros2-foundations',
        'module-1/chapter-2-communication',
        'module-1/chapter-3-urdf',
      ],
    },
    {
      type: 'category',
      label: 'Module 2: The Digital Twin (Gazebo & Unity)',
      link: {
        type: 'generated-index',
        description: 'Learn digital twin concepts, Gazebo physics, and Unity visualization.',

        'module-2/chapter-1-digital-twins',
        'module-2/chapter-2-gazebo-physics',
```

## Troubleshooting

### Problem: Module 2 doesn't appear in sidebar

**Solution**:
1. Verify `book_frontend/docs/module-2/_category_.json` exists
2. Check `sidebar_position` in chapter files (1, 2, 3)
3. Restart dev server: `npm run clear && npm start`

### Problem: Navigation links broken

**Solution**:
1. Verify file names match sidebar config (e.g., `chapter-1-digital-twins.md` not `chapter-1.md`)
2. Check front matter `sidebar_position` matches file order
3. Restart dev server
